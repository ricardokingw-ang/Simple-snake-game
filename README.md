# 🐍 Snake Game (C++ Console)

A classic Snake Game implementation in C++ for the Windows console. Simple, nostalgic, and fun.

![C++](https://img.shields.io/badge/C%2B%2B-11-blue)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)
![License](https://img.shields.io/badge/License-MIT-green)

## 🎮 Gameplay

Control the snake `O` to eat the fruit `@`. Each fruit increases your score by 10 points and makes the snake longer. The game ends if you hit the wall or bite your own tail.

| Key | Action |
|-----|--------|
| `W` | Move Up |
| `S` | Move Down |
| `A` | Move Left |
| `D` | Move Right |
| `X` | Exit Game |
